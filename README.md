# Moe Zay Kha – React 3D Portfolio

This repo has been migrated from a single HTML + plain JS app to a production‑ready Vite + React project with:

- React 18, styled-components, AOS
- @react-three/fiber + @react-three/drei for 3D
- ESLint (React + a11y) + Prettier

Scripts:

- `npm run dev` – local dev
- `npm run build` – production build
- `npm run preview` – preview build
- `npm run lint` – lint
- `npm run format` – format code

See README.REACT.md for structure, migration notes, and deployment.
